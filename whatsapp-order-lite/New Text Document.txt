A lightweight WooCommerce plugin that adds an "Order via WhatsApp" button to product pages, allowing vendors to set their own WhatsApp numbers during registration or from their account page. Fully compatible with vendor plugins like Dokan.

== Description ==

- Adds an "Order via WhatsApp" button to each WooCommerce product.
- Allows each vendor to set a personal WhatsApp number.
- Supports custom registration fields for vendors (Shop Name and WhatsApp Number).
- Fully compatible with WooCommerce and Dokan.
- Vendors can update their WhatsApp number from their profile or vendor dashboard.
- Shows an alert to vendors if no WhatsApp number is set.
- Fallback to a default WhatsApp number if vendorâ€™s number is missing.

1. Go to **WhatsApp Order** in the WordPress admin menu to manage vendor details (Shop Name & WhatsApp Number).
2. Ensure vendors add their WhatsApp number via their account page or vendor dashboard.
3. You can set a default WhatsApp number in the plugin code if no number is provided.
